# Salesforce Integration

Connect Aventir to Salesforce to automate CRM workflows.

## Setup

1. In Salesforce, create a Connected App
2. Enable OAuth and appropriate scopes
3. Generate security token
4. In Aventir, add Salesforce credentials

## Capabilities

- Manage leads and contacts
- Update opportunities
- Track activities
- Generate reports
- Sync data

## Rate Limits

Varies by Salesforce edition. API limits apply.
