Connect Aventir to Slack to send notifications, read messages, and automate team communication.

## Overview

The Slack integration enables agents to:
- Send messages to channels and DMs
- Read channel messages
- Upload files and attachments
- Add reactions
- Create threaded replies

## Setup

1. Navigate to **Settings** â†’ **Integrations**
2. Find **Slack** and click **Connect**
3. Choose your Slack workspace
4. Review permissions
5. Click **Allow**

## Capabilities

### Sending Messages

```
"Send a message to #general that the daily report is ready"
```

### Reading Messages

```
"Check the #alerts channel for urgent messages"
```

### File Sharing

```
"Upload the daily report PDF to #reports"
```

## Common Workflows

- Daily report notifications
- Alert routing from other systems
- Approval workflows
- Team standup reminders

## Security

- OAuth-based authentication
- Scoped permissions only
- Can be revoked anytime

## Rate Limits

Slack tier determines limits. Aventir respects API boundaries automatically.
