Connect Aventir to Discord to send messages and monitor channels.

## Overview

Discord integration enables:
- Send messages to channels
- Read channel messages
- Upload files
- Manage webhooks
- Moderate content

## Setup

1. Go to Discord Developer Portal
2. Create application and bot
3. Generate bot token
4. Invite bot to server
5. In Aventir, add bot token

## Capabilities

- Post to text channels
- Read message history
- Send direct messages
- Upload attachments
- React to messages

## Workflows

- Server notifications
- Event announcements
- Moderation alerts
- Community engagement

## Rate Limits

5 requests per 5 seconds per channel.
