# Shopify Integration

Connect Aventir to Shopify to automate orders, inventory, and store operations.

## Setup

1. In Shopify Admin, go to **Settings** → **Apps** → **Develop apps**
2. Create private app "Aventir"
3. Enable scopes: `read_orders`, `read_products`, `read_inventory`
4. Install app and copy Admin API token
5. In Aventir, add Shopify credentials

## Capabilities

- Monitor and analyze orders
- Track inventory levels
- View product information
- Access customer data

## Workflows

- High-value order alerts
- Low stock notifications
- Daily sales reports
- Abandoned cart recovery

## Rate Limits

40-500+ API calls/second depending on Shopify plan.
