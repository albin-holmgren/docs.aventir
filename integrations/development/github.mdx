Connect Aventir to GitHub to automate development workflows.

## Overview

GitHub integration enables:
- Monitor repositories and issues
- Track pull requests
- Manage releases
- Automate code review
- Sync with project management tools

## Setup

1. Go to GitHub → Settings → Developer settings → Personal access tokens
2. Generate token with repo, issues, pull requests scope
3. In Aventir, go to **Settings** → **Integrations** → **GitHub**
4. Add token and configure

## Capabilities

- Read issues and PRs
- Create comments
- Manage labels
- Check repository status
- View commit history

## Workflows

- Issue triage and labeling
- PR review notifications
- Release note generation
- Dependency monitoring
- Security alert routing

## Rate Limits

5000 requests per hour for authenticated users.
