The AI Manager is your conversational interface to Aventir. Powered by Claude Opus, it understands natural language requests and translates them into autonomous agent deployments.

## Starting a Conversation

Access the AI Manager through the web app at [app.aventir.io](https://app.aventir.io). The interface resembles modern chat applications with:

- Message history with context preservation
- Real-time streaming responses
- File and image attachments (coming soon)
- Voice mode for hands-free interaction

### Your First Message

Introduce yourself and describe what you'd like to automate:

```
"I'm a marketing manager at a SaaS company. I need help with 
social media monitoring, competitor research, and lead tracking."
```

The AI Manager will ask clarifying questions to understand:

- Which tools you use (LinkedIn, Twitter, HubSpot, etc.)
- Your workflow preferences
- Required credentials
- Desired frequency (one-time, daily, weekly)

## Creating Agents Through Chat

The primary way to create agents is by describing your task:

### Simple Task

```
"Check my Gmail for emails from john@company.com and summarize 
what he needs."
```

The AI Manager will:
1. Detect this requires Gmail access
2. Check if you have Gmail credentials stored
3. Create a one-time agent to complete the task
4. Report results in the conversation

### Complex Workflow

```
"Every Monday morning, I need to:
1. Check Shopify for weekend orders
2. Create a summary in Google Sheets
3. Post low-stock alerts to Slack
4. Email the report to my team"
```

The AI Manager will:
1. Break down into component tasks
2. Identify required integrations (Shopify, Google Sheets, Slack)
3. Propose a scheduled workflow
4. Create credentials if needed
5. Deploy agents with dependencies

When you request automation, the AI Manager responds with a plan:

```

**Agent 1: Shopify Order Collector**
- Runs every Monday at 8 AM

**Agent 2: Report Generator**

- Credits: 150 per run

Required: Shopify, Google, Slack credentials
```

## Personalization

The AI Manager adapts to your communication style:

### Detected Preferences

- **Communication style**: Formal vs. casual
- **Detail level**: Brief summaries vs. comprehensive reports
- **Decision approach**: Ask for confirmation vs. act autonomously
- **Response timing**: Immediate vs. scheduled digests

### Setting Preferences

You can explicitly set preferences:

```
"Please keep responses brief and only ask for confirmation 
on tasks costing more than 500 credits."
```

## Best Practices

### Clear Task Descriptions

**Good:**
```
"Check my Gmail inbox every morning at 8 AM, find emails from 
clients with 'urgent' in the subject, and send me a Slack 
notification with the sender and subject line."
```

**Less Effective:**
```
"Check my email"
```

### Provide Context

Share relevant background information:

```
"I'm the VP of Sales. I need to track our HubSpot pipeline 
every Friday at 5 PM and share the forecast with the exec team."
```

### Iterate and Refine

Agents improve with feedback:

```
"That report was good, but can you also include deals that 
moved stages this week?"
```

## Troubleshooting

### Agent Not Created

If the AI Manager can't create an agent:

1. **Check credentials** - Required services may need authentication
2. **Clarify the task** - Provide more specific instructions
3. **Check credit balance** - Insufficient credits prevent deployment
4. **Review limits** - You may have reached agent or task limits

### Unexpected Results

If an agent produces unexpected output:

1. **Describe the issue** - "The report included refunded orders"
2. **Specify corrections** - "Exclude orders with status 'refunded'"
3. **The AI Manager updates** - The agent configuration will be refined

### Conversation Lost

If you need to reference a previous conversation:

```
"Last week you created an agent to check Shopify. Can you 
modify it to also check inventory levels?"
```

## Voice Mode (Beta)

2. Speak naturally to describe tasks
3. The AI Manager transcribes and responds via text
4. Use voice for quick agent creation while mobile
