# Credentials API

Manage service credentials programmatically.

## List Credentials

```bash
GET /v1/credentials
```

Returns credential metadata (values never exposed).

## Add Credential

```bash
POST /v1/credentials
```

Request:
```json
{
  "service": "shopify",
  "type": "api_key",
  "data": {
    "shop": "my-store.myshopify.com",
    "api_key": "xxxxxxxx"
  }
}
```

## Update Credential

```bash
PATCH /v1/credentials/{id}
```

## Delete Credential

```bash
DELETE /v1/credentials/{id}
```

## Supported Services

- Gmail, Slack, Shopify, HubSpot, Notion, OpenAI, and more.
