# Skills Registry

The Skills Registry contains 21 pre-built skills for common business applications.

## Skill Structure

Each skill includes:
- **Instructions**: Best practices for using the service
- **Common Tasks**: Typical workflows and patterns
- **Known Issues**: Documented problems and solutions
- **Credential Requirements**: Required access
- **Risk Level**: Low, medium, or high

## Skill Categories

| Category | Services |
|----------|----------|
| Email | Gmail, Outlook |
| Social | LinkedIn, Twitter |
| Productivity | Notion, Google Workspace |
| Communication | Slack, Discord |
| CRM | HubSpot, Salesforce |
| E-commerce | Shopify |
| Development | GitHub |
| AI | OpenAI |

## Dynamic Detection

The Dynamic Agent Builder detects relevant skills from task descriptions.

## Risk Levels

- **Low**: Read-only operations
- **Medium**: Send messages, create content
- **High**: Financial transactions, data deletion
